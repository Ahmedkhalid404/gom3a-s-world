Pages Description

Vs Page

In this page, you enter two handles and it gives you some data for each one. Then, it shows you all the problems solved by the first handle, categorized by rating. You can check for each problem whether the second person has solved it or not by a mark next to each problem. You can also get a random problem from the first person's solved problems with a specific rating, and ensure that the second person has not solved it.

Make contest (MK contest) Page

In this page, you can enter any number of handles you like, separated by spaces, commas, or line breaks, and then specify the number of problems you want (range from 1 to 100). You will receive problems from each rating range between 800 and 3500. You’ll get problems in each rating with the number you specified, and all the handles will not have solved them.

Game Page

In this page, it's just a game I enjoy playing with myself, and I wanted to share it with you. I try to guess the rating of a problem, so I’ll give you a random problem (and you’ll determine whether it’s one of the problems you’ve solved or not). You read it and try to guess its rating. I tried to let you choose whether you want the problem's statement or to get a random accepted code for the problem so you can try to guess its rating from the code, but I couldn't find a way to access the Codeforces API to get the code. :))
